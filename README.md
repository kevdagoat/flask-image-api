# ImageAPI
### A Flask RESTPlus API for storing and resizing images

## Installation

To use, please install Python 3 and run

```python
pip3 install -r requirements.txt
```

## Usage

Run 

```
python3 main.py 
```

And browse to [http://localhost:8000](http://localhost:8000) for the Swagger GUI.
